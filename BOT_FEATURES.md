### 1️⃣ **КАТАЛОГ** - Перегляд та Вибір Кави

#### Що можна робити?
- **Переглядати каву** за форматом (300г або 1кг)
- **Фільтрувати** за способом приготування:
  - ☕ **Espresso** - для кавоварок
  - 🌊 **Filter** - для фільтра, Аеропреса
  - 🔄 **Universal** - для всього
- **Переглядати детальну інформацію** про кожну каву:
  - Профіль смаку
  - Рівень обсмажування
  - Описи ароматів
  - Ціна
- **Переглядати списки** - організовані колекції товарів

#### Як користуватись?
```
Головне меню → 🛍️ Каталог
  ↓
Вибір формату (300г або 1кг)
  ↓
Вибір профілю (Espresso/Filter/Universal)
  ↓
Вибір конкретної кави з переліку
  ↓
Прочитати опис → Додати до кошика
```

#### Новий розділ: **🌳 Списки**
```
Головне меню → 🌳 Списки
  ↓
Перегляд ієрархічної структури:
🏠 Головне меню
├── ☕ Кава
│   ├── 🎯 Для Еспресо
│   └── 🎯 Для Фільтру

└── 🎁 Колекції
```

1. **Економія часу адміністратора** - Автоматичне генерування описів товарів
2. **Консистентність контенту** - Однорідний стиль описів
3. **Масштабованість** - Можливість швидко створювати контент для нових товарів
4. **Персоналізація** - AI-рекомендації списків на основі аналітики
5. **Підвищення конверсії** - Якісні маркетингові описи залучають клієнтів

## 💰 Система Розрахунків

### Порядок обчислення знижок:

```
1. Вихідна ціна              500 грн

2. Обсяг замовлення:
   Якщо 7+ упаковок (300г) або 2+ кг → -25%
   Розрахунок: 500 × 0.75 = 375 грн

3. Лояльність (від рівня):
   Bronze (-5%)  → 375 × 0.95 = 356.25 грн
   Silver (-10%) → 375 × 0.90 = 337.50 грн
   Gold (-15%)   → 375 × 0.85 = 318.75 грн

4. Промо-код:
   Якщо є → найбільша знижка застосовується

5. Безкоштовна доставка:
   Якщо замовлення ≥ 1500 грн → безкоштовно

ПРИКЛАД:
┌─────────────────────────────────┐
│ 3 упаковки × 300г = 1200 грн    │
├─────────────────────────────────┤
│ Знижка за обсяг (-25%): -300 грн │
│ Лояльність Gold (-15%): -135 грн│
│ Промо SAVE10: -120 грн           │
├─────────────────────────────────┤
│ ВИБИРАЄТЬСЯ: -300 (найбільша)    │
├─────────────────────────────────┤
│ Підсумок: 1200 - 300 = 900 грн  │
│ Доставка: +80 грн               │
│ ВСЬОГО: 980 грн                 │
│ Заощаджено: 300 грн             │
└─────────────────────────────────┘
```
