# Стильгайд — Оливково-Оранжева Тема

Мета: один візуальний стиль для бота та супутніх матеріалів. Основні кольори — оливковий (primary) і оранжевий (accent).

## Палета кольорів
- **Оливковий — primary**: `#6B8E23` (Olive Drab)
- **Оливковий темний**: `#53661B`
- **Оливковий світлий**: `#9FB86B`
- **Оранжевий — accent**: `#FF7A00`
- **Оранжевий темний**: `#E66900`
- **Нейтральний фон**: `#FAF9F6`
- **Темний текст**: `#1F1F1F`
- **Світлий текст**: `#FFFFFF`

### Контраст та доступність
- Використовуйте достатній контраст між текстом і фоном. Для кнопок з оливковим фоном — текст білий або дуже темний.
- Для невеликих елементів (іконок, дрібного тексту) перевіряйте контраст з інструментами WCAG.

## Типографіка
- Основний шрифт: системний інтерфейс або `Inter, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial`.
- Розміри (орієнтовно):
  - Заголовок H1: 28px
  - H2: 22px
  - H3: 18px
  - Тіло: 14-16px
  - Малий текст: 12px

## Компоненти
- Кнопка основна (`.btn-primary`): фон — оливковий, текст — білий, округлі кути 8px.
- Кнопка акцент (`.btn-accent`): фон — оранжевий, текст — білий, припіднята тінь.
- Картки (`.card`): фон білий/нейтральний, легка тінь, padding 16px.
- Бейджі (`.badge`): оранжеві для підсвітки, оливкові для статусів.

## Іконки та кнопки
- Використовуйте прості, монохромні SVG іконки; для важливих дій додатково підкреслюйте кольором акценту.
- В SVG рекомендовано використовувати `currentColor` для заливки, щоб можна було контролювати колір через CSS.

## Приклади використання
- Стартовий екран: фон нейтральний `#FAF9F6`, головний CTA — оранжева кнопка.
- Меню/клавіатура бота: кнопки в оливкових відтінках з білим текстом.

## Файли та реалізація
- CSS-тема: `assets/styles/theme.css`
- SVG-шаблони кнопок: `assets/images/button_olive.svg`, `assets/images/button_orange.svg`

## Поради
- Дотримуйтесь однієї системи відступів (8px модуль).
- Не комбінуйте більше 2-х основних кольорів на одному екрані.

Якщо хочете, можу додати кольорові зразки у форматі PNG або згенерувати SCSS-змінні та приклади компонентів у `src/`.
